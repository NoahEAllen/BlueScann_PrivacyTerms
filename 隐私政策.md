## 一、我们如何收集和使用您的个人信息

### 1.1 功能说明

BlueScann 是一款蓝牙设备扫描工具，主要功能包括：
- 扫描附近的蓝牙设备
- 显示蓝牙设备的信号强度、距离等信息
- 保存和管理已发现的蓝牙设备
- 提供设备位置信息（如适用）

### 1.2 我们收集的信息

#### 1.2.1 设备信息
- **蓝牙设备信息**：当您使用扫描功能时，我们会收集附近蓝牙设备的以下信息：
  - 设备名称
  - 设备 MAC 地址（虚拟地址）
  - 设备信号强度（RSSI）
  - 设备类型（耳机、手表、手机、音响等）
  - 设备发现时间

- **位置信息**：为了提供更准确的设备距离估算，我们可能会请求您的位置权限：
  - 精确位置信息（如您授权）
  - 模糊位置信息（如您授权）

#### 1.2.2 应用使用信息
- **设备列表数据**：您保存的蓝牙设备列表信息，仅存储在您的设备本地
- **应用设置**：您的应用偏好设置（如排序方式、显示单位等），仅存储在您的设备本地

### 1.3 我们如何使用您的信息

我们使用收集的信息用于以下目的：
1. **提供核心功能**：扫描、显示和管理蓝牙设备
2. **改善用户体验**：根据您的设置偏好提供个性化服务
3. **数据存储**：在您的设备本地存储设备列表和设置信息

### 1.4 权限使用说明

本应用需要以下权限以实现相关功能：

| 权限名称 | 用途说明 | 是否必需 |
|---------|---------|---------|
| **蓝牙权限** | | |
| `ohos.permission.ACCESS_BLUETOOTH` | 扫描和发现附近的蓝牙设备 | 必需 |
| `ohos.permission.DISCOVER_BLUETOOTH` | 配置蓝牙并查找远端设备 | 必需 |
| `ohos.permission.MANAGE_BLUETOOTH` | 配对和管理蓝牙设备 | 必需 |
| **位置权限** | | |
| `ohos.permission.LOCATION` | 获取精确位置，用于设备距离估算 | 可选 |
| `ohos.permission.APPROXIMATELY_LOCATION` | 获取模糊位置，用于设备距离估算 | 可选 |

**重要说明**：
- 所有权限均用于实现应用的核心功能
- 位置权限为可选权限，拒绝授权不会影响基本的蓝牙扫描功能
- 所有数据仅存储在您的设备本地，不会上传到服务器

## 二、数据存储和安全

### 2.1 数据存储位置

- **本地存储**：所有收集的数据（设备列表、设置信息等）均存储在您的设备本地，使用系统提供的 Preferences 存储机制
- **云端存储**：我们**不会**将您的任何个人信息上传到云端服务器

### 2.2 数据保留期限

- 设备列表和设置信息将保留在您的设备上，直到您：
  - 手动删除应用数据
  - 卸载应用
  - 清除应用缓存

### 2.3 数据安全措施

我们采用以下安全措施保护您的信息：
- 使用系统提供的安全存储机制
- 所有数据仅在您的设备本地处理
- 不进行任何网络传输

### 3.1 信息共享

### 3.2 第三方服务

根据相关法律法规，您对自己的个人信息享有以下权利：

### 4.1 访问权
您有权访问我们收集的关于您的个人信息，包括：

- 查看应用设置

### 4.2 删除权
您可以随时：

- 清除应用数据

### 4.3 撤回同意权
您可以随时在系统设置中：
- 撤回位置权限授权
- 撤回蓝牙权限授权（将影响应用功能）

如果您认为我们的个人信息处理行为损害了您的合法权益，您可以向相关监管部门投诉举报。
